# About
This is a beginner's GOlang project.
I have written a program using which you can find some of the best anime in each category.

# Objective 
* Learn Go lang and build project as a beginner.
* Enhance Go skills and learnings by building project. 

# Prerequisite
You should have basic knowledge of Go language like data types, if-else statement, loops, switch-statement, slices, maps and structures.

# Overview
Want to watch anime!? Don't know what to watch? I got you you covered.

* Golang basics
  - Fmt (Printf and Scan) 
  - Variables and Data types
  - Type Conversion
  - String padding
  - Conditionals(if/else)
  - Switch Statement
  - Arrays and Slices 
  - Maps
  - Structs
